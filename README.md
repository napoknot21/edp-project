
# FEM Project

New clean project for the MU4MA329 project.

Run:

```bash
pip install -r requirements.txt
python main.py --mesh path/to/rectangle01.msh --p 1 --q 1 --bx 1 --by 1 --c 1 --plot
```

Convergence mode:

```bash
python main.py --mode convergence --meshes mesh1.msh mesh2.msh mesh3.msh --p 1 --q 1
```
